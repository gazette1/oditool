§ 02 · The process 02 / 05

Five passes. Compounding rigor.

Each pass narrows the search space and is independently verifiable. You can re-run any pass in isolation, audit its prompt, and override its output before the next pass consumes it. No black box.

Pass 01
Discover core jobs
Identify five core functional jobs in the sector. Verb + object + contextual clarifier, solution-agnostic, stable over time. Plus job executor, related jobs, emotional jobs, social jobs, consumption-chain jobs, and seed search queries.
→ 5 jobs · 50 sub-jobs
Pass 02
Map the universal steps
Each job decomposed into 8 process steps: Define, Locate, Prepare, Confirm, Execute, Monitor, Modify, Conclude. Five desired outcomes per job, scored on importance and current satisfaction. Run one job at a time so nothing truncates.
→ 25 outcomes · scored
Pass 03
Validate against demand
Each job validated against live search behavior using Claude's web-search tool. Returns a 0–100 demand signal, rising/stable/declining trend, competition density, and a top-keyword snapshot.
→ 5 demand reports
Pass 04
Rank entry strategies
Synthesizes scored data into 3–5 ranked market entry strategies using Ulwick's Growth Strategy Matrix: Differentiated, Dominant, Disruptive, Discrete, Sustaining. Each rec includes a first-move experiment and a key risk.
→ 5 entry strategies

§ 05 · Methodology 05 / 05

Built on Ulwick's
Outcome-Driven Innovation.

ODI is not jobs-to-be-done. JTBD is the framing; ODI is the operating system. Tony Ulwick's framework defines outcomes as measurable, controllable, and stable over time, and scores them with the opportunity algorithm:

Opportunity = Importance + max(Importance − Satisfaction, 0)

Ulwick · Strategyn · 1991

Every outcome the Engine produces is in the canonical Ulwick format: Minimize the [time / likelihood] [it takes to / that] [object of control] [contextual clarifier]. No buzzwords. No lifestyle copy. The format is what makes the output composable across passes — and what lets a board read the artifact without translation.
